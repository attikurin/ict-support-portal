ダウンロード資料

ICT支援員の業務に役立つチェックリスト、テンプレート、ワークシートを無料でダウンロードできます。印刷してご活用ください。

ご利用について: 各資料は教育現場での使用を目的として提供しています。商用利用はご遠慮ください。 印刷またはPDF保存は、ブラウザの印刷機能（Ctrl+P / Cmd+P）をご利用ください。

チェックリスト

セキュリティチェックリスト

学校ICTセキュリティの定期点検用チェックリスト。アクセス管理、端末管理、ネットワーク、データ管理、教育、インシデント対応の6カテゴリー・30項目以上を網羅。

  • ✓ 優先度別（高・中・低）の項目分類
  • ✓ チェック状態と備考欄付き
  • ✓ 総合評価セクション
開く・ダウンロード

テンプレート・ワークシート

授業支援計画シート

効果的な授業サポートのための事前計画シート。授業概要、使用機器、支援内容、トラブル対応策、振り返りを一元管理。

  • ✓ 授業前・中・後の支援内容記入欄
  • ✓ 想定トラブルと対応策
  • ✓ 配慮が必要な児童生徒情報
開く・ダウンロード

研修設計テンプレート

教職員向けICT研修の企画・実施用テンプレート。研修概要、対象者分析、プログラム設計、評価方法まで体系的に整理。

  • ✓ 時間配分とプログラム詳細
  • ✓ 準備物・資料リスト
  • ✓ フォローアップ計画
開く・ダウンロード

自己評価シート

ICT支援員のスキルアップと成長のための振り返りツール。技術、授業支援、コミュニケーション、専門知識、業務遂行力の5領域・20項目以上。

  • ✓ 5段階評価（1-5）
  • ✓ 領域別スキル評価
  • ✓ 成長記録と目標設定欄
開く・ダウンロード

活用のポイント

印刷・PDF保存方法

  1. 各資料のリンクをクリックして新しいタブで開く
  2. ブラウザの印刷機能を使用（Ctrl+P / Cmd+P）
  3. 「送信先」でプリンターまたは「PDFに保存」を選択
  4. 必要に応じて印刷設定を調整して保存・印刷

活用のコツ

  • 定期的な活用: チェックリストは月1回など定期的に実施
  • チーム共有: 計画シートやテンプレートをチームで共有
  • カスタマイズ: 学校の実情に合わせて項目を追加・修正
  • 記録保存: 記入済みシートは業務記録として保管

よくある質問

Q: 商用利用は可能ですか？
A: 教育現場での使用に限ります。商用利用はご遠慮ください。
Q: 内容を編集できますか？
A: 印刷後に手書きで追記いただけます。PDF編集ソフトでも編集可能です。
Q: 更新予定はありますか？
A: 定期的に内容を見直し、必要に応じて更新します。
