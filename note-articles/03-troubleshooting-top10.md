## 🥇 第1位: 端末がWi-Fiに繋がらない

### 症状
- 「インターネットに接続できません」と表示
- Wi-Fiマークに「×」がついている
- 他の端末は繋がっているのに、特定の端末だけ繋がらない

### よくある原因
1. **Wi-Fiの電波が弱い**
   - 教室がアクセスポイントから遠い
   - 鉄筋コンクリートの壁で電波が届かない
2. **接続台数の上限に達している**
   - 1つのアクセスポイントに40台以上接続
3. **IPアドレスの競合**
   - DHCP（自動IPアドレス割り当て）の不具合
4. **端末の設定ミス**
   - 間違ったSSIDに接続している
   - パスワード入力ミス

### 解決方法（優先度順）

#### ⚡ 即効対応（30秒）
```
1. 機内モードをON→OFFにする
2. 端末を再起動する
3. Wi-Fi設定画面で「接続済み」のネットワークをタップ→「削除」→再接続
```

#### 🔧 それでもダメなら（3分）
```
4. 近くのアクセスポイントを探す
   - 設定→Wi-Fi→利用可能なネットワーク一覧を確認
   - 電波強度が強いものに接続
5. 別の場所（廊下、職員室近く等）で試す
6. 他の端末も同じ症状か確認
   - 他も繋がらない→ネットワーク全体の問題
   - 特定の端末だけ→端末固有の問題
```

#### 🛠️ 根本対応（時間があるとき）
```
7. アクセスポイントの増設を検討
8. チャンネル設定の最適化
9. MDMで端末の設定を一括配信
```

### 📝 予防策
- 授業前にWi-Fi接続テストを実施
- 電波強度マップを作成（廊下を歩いて電波チェック）
- 接続台数が多い時間帯を把握

## 🥈 第2位: ログインできない（パスワード忘れ）

### 症状
- 「パスワードが違います」と表示
- アカウントがロックされた
- 児童生徒が泣きそうになっている

### よくある原因
1. **パスワードを忘れた**
   - 夏休み明け、冬休み明けに多発
2. **入力ミス**
   - 大文字小文字の間違い
   - 全角半角の間違い
   - 数字の「0（ゼロ）」と「O（オー）」を間違える
3. **アカウントロック**
   - 3回以上間違えて入力した

### 解決方法（優先度順）

#### ⚡ 即効対応（1分）
```
1. 入力方法を確認
   - Caps Lock がONになっていないか
   - 日本語入力がONになっていないか（半角英数に切替）
   - テンキーのNum Lockを確認
2. パスワード表示機能を使う
   - 「目」のアイコンをタップしてパスワードを表示
   - 何を入力しているか確認
```

#### 🔧 それでもダメなら（5分）
```
3. Google Workspace / Microsoft 365の管理画面でパスワードリセット
   - admin.google.com にアクセス
   - ユーザー検索→パスワードをリセット
4. アカウントロックを解除
   - 管理画面から「ロック解除」ボタンをクリック
```

#### 🛠️ 根本対応
```
5. パスワードを児童生徒が覚えやすいものに変更
   - 例: 学年+クラス+出席番号+学校名の頭文字
   - 5-1-15-H（5年1組15番、○○小学校）
6. パスワードカードを作成して配布
   - 紛失防止のため連絡帳に貼る
```

### 📝 予防策
- 学期の始めに全員でログイン練習
- パスワードリセット手順を教職員に共有
- 管理者アカウントを複数人で共有（引き継ぎミス防止）

## 🥉 第3位: 音が出ない

### 症状
- 動画を再生しても音が聞こえない
- マイクが反応しない
- イヤホンをしても音が出ない

### よくある原因
1. **ミュートになっている**
   - ソフトウェア側でミュート
   - 本体側でミュート
2. **音量がゼロ**
   - ボリュームスライダーが最小
3. **イヤホンが刺さったまま**
   - スピーカーから音が出ない設定
4. **Bluetooth接続が残っている**
   - 前回使ったBluetoothスピーカーに接続されている

### 解決方法（優先度順）

#### ⚡ 即効対応（30秒）
```
1. ミュートボタンを確認
   - 画面右下のスピーカーアイコンをクリック
   - ミュート解除
2. 音量を上げる
   - ボリュームスライダーを右に動かす
   - 本体側のボリュームボタンも確認
3. イヤホンジャックを確認
   - イヤホンが刺さっていないか確認
   - 刺さっている場合は抜く
```

#### 🔧 それでもダメなら（2分）
```
4. 出力先を確認
   - 設定→サウンド→出力デバイス
   - 「内蔵スピーカー」または「イヤホン」を選択
5. Bluetooth接続を解除
   - 設定→Bluetooth→接続済みデバイスを削除
6. 端末を再起動
```

#### 🛠️ ハード的な問題の場合
```
7. スピーカーの故障を疑う
   - イヤホンで音が出るか確認
   - 出る→スピーカー故障、出ない→他の原因
8. 修理または交換を検討
```

### 📝 予防策
- 授業前に音声テスト
- Bluetooth使用後は必ず解除
- イヤホンジャックにホコリが溜まらないようカバー

## 4位: プロジェクターに映らない

### 症状
- 「No Signal」と表示
- 画面が真っ黒
- PCの画面は映るがタブレットが映らない

### よくある原因
1. **入力切替ができていない**
   - HDMI1、HDMI2、VGA等の切替ミス
2. **ケーブルの接続不良**
   - HDMIケーブルが緩んでいる
   - 断線している
3. **端末の設定ミス**
   - ミラーリング設定がOFFになっている

### 解決方法

#### ⚡ 即効対応（1分）
```
1. 入力切替ボタンを押す
   - プロジェクターのリモコン or 本体ボタン
   - HDMI1、HDMI2、VGA等を順番に試す
2. ケーブルを抜き差しする
   - しっかり奥まで差し込む
3. 端末を再起動
```

#### 🔧 それでもダメなら（3分）
```
4. 別のケーブルを試す
5. 別の端末で映るか確認
   - 映る→元の端末の問題
   - 映らない→プロジェクターorケーブルの問題
6. Windows: Win + P キーで投影モード変更
   Mac: システム環境設定→ディスプレイ→ミラーリング
```

## 5位: 印刷できない

### 症状
- 印刷ボタンを押しても印刷されない
- 「プリンターがオフライン」と表示
- 白紙が出てくる

### よくある原因
1. **プリンターがオフライン**
   - 電源が入っていない
   - ネットワーク接続が切れている
2. **用紙切れ/インク切れ**
3. **印刷ジョブの詰まり**
   - 前の印刷が残っている

### 解決方法

#### ⚡ 即効対応（2分）
```
1. プリンターの電源・接続確認
   - 電源ランプが点灯しているか
   - Wi-Fi/LANケーブルが接続されているか
2. 用紙・インクの確認
   - 用紙を補充
   - インクカートリッジを交換
3. 印刷キューのクリア
   - 設定→デバイス→プリンターとスキャナー
   - 印刷ジョブを削除
```

## 6位: Classroomで課題が配れない

### 症状
- 「権限がありません」と表示
- 生徒が課題を見られない
- ファイルが添付できない

### 解決方法
```
1. 共有設定を確認
   - ファイルの共有権限を「リンクを知っている全員」に変更
2. Classroomのクラスメンバーを確認
   - 生徒が正しく登録されているか
3. ストレージ容量を確認
   - Googleドライブの容量が足りているか
```

---

## 7位: Teamsの会議に入れない

### 症状
- 「会議に参加できません」と表示
- カメラやマイクが使えない
- 画面共有ができない

### 解決方法
```
1. ブラウザのキャッシュをクリア
2. カメラ・マイクの許可設定を確認
   - 設定→プライバシー→カメラ/マイク
3. Teamsアプリを最新版に更新
```

---

## 8位: 端末の動作が遅い

### 症状
- アプリの起動が遅い
- 画面がカクカクする
- タッチ操作の反応が悪い

### 解決方法
```
1. 不要なタブ・アプリを閉じる
2. 端末を再起動
3. ストレージ容量を確認
   - 不要なファイルを削除
4. キャッシュをクリア
```

---

## 9位: ファイルが開けない

### 症状
- 「このファイルは開けません」と表示
- ダウンロードできない
- 文字化けする

### 解決方法
```
1. ファイル形式を確認
   - 対応するアプリがインストールされているか
2. 共有権限を確認
   - ファイルの閲覧権限があるか
3. 別のアプリで開く
   - Googleドライブのプレビュー機能を使う
```

---

## 🔟 第10位: アップデートが終わらない

### 症状
- 「アップデート中」のまま固まる
- 再起動を繰り返す
- 画面が真っ暗

### 解決方法
```
1. 15分以上待つ（特にWindows Update）
2. 強制再起動
   - 電源ボタン長押し（10秒）
3. セーフモードで起動
4. 最悪の場合は初期化
   - データバックアップを確認してから実行
```

## 📝 トラブル対応の心得

### 1. **焦らない**
先生も児童生徒も焦っています。あなたまで焦ると収拾がつきません。深呼吸して落ち着いて対応。

### 2. **記録を残す**
対応したトラブルは必ずメモ。次回同じトラブルが起きた時に即対応できます。

### 3. **再現性を確認**
同じ操作で再現するか確認。再現すればメーカーサポートに連絡しやすい。

### 4. **エスカレーション基準を決める**
15分以内に解決できなければ、業者やメーカーに連絡。一人で抱え込まない。

### 5. **予防が最重要**
トラブルが起きてから対応するより、事前に防ぐ方が100倍楽。
