# ICT支援員総合ポータルサイト

教育現場でICT支援に携わる全ての方へ、実践的な情報とリソースを提供する総合ポータルサイトです。

## 🎯 プロジェクト概要

このサイトは、学校現場でICT活用を支援する方々のために、以下の情報を包括的に提供します：

- **🔍 サイト横断検索機能** - 全ページを即座に検索できる検索ボックスをヘッダーに統合 ⭐NEW (2025/12)
- **📥 ダウンロード資料** - チェックリスト・テンプレート・ワークシート（セキュリティ、授業支援、研修、自己評価）⭐NEW (2025/12)
- **🤖 AI搭載教育アプリ・ツール総合ガイド** - AIドリル、生成AI、学習支援AI、語学AI、特別支援AIを網羅 ⭐NEW (2025/12)
- **💻 実務・技術支援ガイド3本** - 端末管理・MDM、Wi-Fiトラブル対応、アカウント管理 ⭐NEW (2025/12)
- ICT支援員の基礎知識と専門用語集
- 効果的な支援ガイドラインとアプローチ
- GIGAスクール構想の最新動向
- **学校でのAI利用ガイド（Google Gemini、Microsoft Copilot、文部科学省ガイドライン）**
- **資格・検定情報（ICT支援員認定試験、ITCE検定、関連資格）**
- **配置・統計データ（全国7,172人配置、配置状況の推移、配置形態）**
- **ICT機器・ツール一覧（学習者用端末、大型提示装置、MDM、授業支援システム等）**
- 全国の成功事例とベストプラクティス
